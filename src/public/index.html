Seconds Metering API

A production-ready metering service that parses your invoice catalog, tracks per-second usage, and generates invoice JSON from session summaries.

Catalog: /catalog Sessions: /sessions Invoice: /invoices/from-session NDSP: /ndsp/state NDSP: /ndsp/telemetry

Quick Test

curl -H "x-api-key: dev-key-1" {BASE}/health
curl -H "x-api-key: dev-key-1" {BASE}/catalog

Replace {BASE} with your server URL (ex: http://localhost:8080).

Security

Use one API key per client/seat. Put this behind HTTPS (Nginx / Caddy / Cloudflare). 

Header: x-api-key: <your-key>
